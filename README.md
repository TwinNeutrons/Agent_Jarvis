# Agent_Jarvis

## 🧠 Overview

Agent_Jarvis is a local-first AI assistant that runs continuously and can be accessed via WhatsApp.

It is designed to evolve into a fully agentic system capable of planning, executing, and automating real-world tasks using natural language.

---

## 🎯 Vision

Build a personal AI system that:
- runs 24/7
- understands context and memory
- uses tools intelligently
- evolves into a modular agent platform

---

## ⚙️ Current Capabilities

- WhatsApp integration (WAHA)
- Local LLM (Ollama)
- Automated startup (Docker + Python)
- Context-aware conversation memory
- Structured logging system
